# 🛡️ Password Checker

A lightweight, logic-based security tool designed to evaluate password strength in real-time. This project focuses on JavaScript string manipulation and DOM interaction to provide instant feedback to the user.

## ✨ Key Features
* **Real-time Validation:** Uses JavaScript event listeners to check strength as the user types.
* **Strength Indicators:** Visual feedback (e.g., color changes or text alerts) based on security levels.
* **Security Criteria:** Checks for essential patterns like:
  * Minimum length (e.g., 8+ characters).
  * Inclusion of numbers.
  * Special character detection (@, #, $, etc.).
* **Clean UI:** A minimalist interface built with semantic HTML5 and custom CSS3.

## 🛠️ Tech Stack
* **Frontend:** HTML5 & CSS3
* **Logic:** JavaScript (ES6+)
* **Version Control:** Git & GitHub
